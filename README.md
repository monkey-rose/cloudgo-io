# cloudgo-io
是一个酒店预定的应用，其中酒店介绍 http://127.0.0.1:8080/static/ 是用了静态页面，其中酒店的地址电话是通过js传递数据，表单提交 http://127.0.0.1:8080/reserve 提交成功后会跳转出一个信息表格
